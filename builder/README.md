### Setting Electron Project with Typescript

Requires [node.js](https://nodejs.org) version `>=20.4.0`

### Install dependencies

```sh
yarn install
```

### Run

```sh
yarn start
```

Path에 환경변수 추가

```sh
C:\windows\system32
```

```mac
export PATH="$PATH:/mnt/c/Windows/System32" npm run dev
```

### Build

```sh
yarn make
```
